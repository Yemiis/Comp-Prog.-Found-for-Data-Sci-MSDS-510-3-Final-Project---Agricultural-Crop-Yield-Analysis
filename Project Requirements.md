# 📘 MSDS 510 – Python Data Analysis Project Requirements

## 📌 Project Overview

This project focuses on applying foundational Python programming concepts to analyze a real-world dataset. The goal is to combine:

- CSV file processing
- Lists and dictionaries
- Object-Oriented Programming (OOP)
- Data analysis techniques

Students are required to explore a dataset, formulate analytical questions, and develop a structured Python program to answer those questions.

According to the project description, the program must:
- Import a CSV dataset
- Process and analyze the data
- Output results to a text or CSV file :contentReference[oaicite:0]{index=0}
